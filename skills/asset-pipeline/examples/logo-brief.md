# Logo Brief

Create a clean SVG logo mark for a developer tooling brand named "Traceform".

- Tone: precise, durable, technical, not playful
- Audience: developers and platform engineers
- Use cases: docs header, GitHub README, favicon-scale mark, dark and light backgrounds
- Avoid: mascots, gradients as the core idea, excessive detail, trendy blobs
- Desirable concepts: path tracing, signal flow, layered structure, trustworthy instrumentation
